Mailing lists

If you want to send patches or if you see for advice that's related to libvirt-php or libvirt you can use following mailing list:

php-virt-control@lists.fedorahosted.org
Archives at https://fedorahosted.org/pipermail/php-virt-control/
This list is a place for discussions about the development of php-virt-control.
Topics for discussion include:
  • New features for php-virt-control
  • Bug fixing of php-virt-control
  • Testing and documentation of php-virt-control
